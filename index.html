Setup Checklist

  • Run php setup.php to initialize database
  • Set admin password with php set-password.php
  • Configure CORS origins in config.php
  • Copy Hugo shortcode to your site
  • Import existing comments (optional)
  • Test posting a comment on example.html

Import Your Comments

Already have comments from another platform? Import them easily:

  • From Disqus: php import-disqus.php export.xml
  • From TalkYard: php import-talkyard.php export.json

API Endpoints

Your comment system exposes these endpoints:

  • GET api.php?action=comments&url={page} - Fetch comments
  • POST api.php?action=post - Submit new comment
  • POST api.php?action=login - Admin login
  • GET api.php?action=pending - Get pending comments (admin)
  • PUT api.php?action=moderate&id={id} - Moderate comment (admin)
